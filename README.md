# 💱 Currency Converter Web App

A simple and responsive Currency Converter web application built using HTML, CSS, and JavaScript.  
It allows users to convert one currency to another using real-time exchange rates.

---

## 🚀 Features

- 🌍 Convert between multiple world currencies
- 🔄 Real-time exchange rates using API
- 📱 Responsive design (Mobile Friendly)
- ⚡ Fast and lightweight

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Exchange Rate API
